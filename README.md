# C-Lyrics

Chinese lyrics powered by AI so anyone can learn from music they love.

C-Lyrics is a community hub for Chinese music fans and language learners worldwide. Search the shared song library, study with pinyin and translation generated by AI, and sync lyrics to YouTube - no account required. Anyone can add songs directly from the app and contribute to the community library via a simple built-in flow backed by GitHub.

## Features
- AI-generated pinyin + translation for any Chinese lyrics
- YouTube video sync with lyric scroll
- Fullscreen single-line mode for screen recording
- Offline support
- Community song library - contribute without leaving the app

## Live
[clyrics.phucbm.com](https://clyrics.phucbm.com)

## Contribute Songs
Open C-Lyrics, load or create a song, hit the PR button. No GitHub account setup needed on your end - just a username and personal access token with `repo` scope. Your song goes straight to the community library.

Want to contribute to the app itself? PRs and issues welcome at [phucbm/clyrics](https://github.com/phucbm/clyrics).

## Dev Setup
```bash
pnpm install
pnpm dev
```

Set `VITE_GROQ_API_KEY` in `.env` or enter it in app Settings.
